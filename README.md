# ToneMate

ToneMate is a simple tone generator UI Frontend based on NAudio `ToneGenerator`.

## Features

**Supported Tones**:

- Sine
- Triangle
- Sawtooth
- Square
- Pink Noise
- Silence

**Other**:

- Preview tones
- Change frequency and duration

## Build

You will need .NET 7 Desktop SDK. Then, open your terminal and run `dotnet build`.
